Curso: Configuración y Protocolo de Red

JYB Netlearn - Formación en redes profesionales

Bienvenida

En este curso aprenderás a configurar, analizar y comprender los protocolos que hacen posible la comunicación en redes de datos. Se abordarán temas que van desde la configuración básica de dispositivos hasta la implementación de protocolos de enrutamiento dinámico y servicios de red esenciales.

El objetivo es que adquieras las competencias necesarias para diseñar, implementar y administrar redes pequeñas y medianas aplicando buenas prácticas y metodologías de diagnóstico. A lo largo del curso se realizarán múltiples laboratorios prácticos utilizando simuladores como Cisco Packet Tracer, GNS3 y EVE-NG, lo que te permitirá desarrollar habilidades aplicables en escenarios reales.

Módulo 1: Configuración inicial de dispositivos de red

En este módulo se explorará cómo acceder a un router o switch, ya sea mediante consola o acceso remoto, y se configurarán parámetros básicos como nombres de host, contraseñas de acceso y mensajes de advertencia. Se realizará la configuración de interfaces de red, incluyendo la asignación de direcciones IP, máscaras de subred y gateways predeterminados.

También se aprenderá a guardar configuraciones en la memoria de inicio y a verificar el correcto funcionamiento de las interfaces con comandos de diagnóstico. El estudiante comprenderá la diferencia entre la configuración en ejecución (running-config) y la configuración guardada (startup-config), así como la importancia de la documentación de cada cambio realizado.

Módulo 2: Direccionamiento IP y subredes

Este módulo se centra en el direccionamiento IP, explicando cómo funciona IPv4, las clases de direcciones, la creación de subredes y el uso de VLSM para optimizar el espacio de direccionamiento. El estudiante aprenderá a planificar esquemas de direccionamiento para diferentes topologías, asignando direcciones a routers, switches y hosts finales.

Se realizarán ejercicios prácticos para identificar errores comunes, como conflictos de direcciones, máscaras incorrectas o gateways mal definidos. Asimismo, se introducirá el concepto de direccionamiento IPv6, destacando sus ventajas frente a IPv4 y las formas de transición entre ambos protocolos.

Módulo 3: Protocolos de enrutamiento

En este módulo se estudiarán los protocolos de enrutamiento dinámico, como RIP, OSPF y EIGRP. Se analizará cómo los routers intercambian información de enrutamiento para encontrar la mejor ruta hacia un destino, y se practicarán configuraciones paso a paso en laboratorios virtuales.

También se abordará el enrutamiento estático y sus diferencias con el dinámico, explicando en qué escenarios es más conveniente cada enfoque. Se evaluará cómo diagnosticar problemas de enrutamiento utilizando herramientas como traceroute, tablas de enrutamiento y depuración de protocolos.

Módulo 4: Protocolos de capa de aplicación

Aquí se explorará cómo funcionan los protocolos que permiten servicios esenciales dentro de una red. Se configurarán servidores y clientes para servicios como DHCP, que asigna direcciones automáticamente, DNS, que traduce nombres a direcciones IP, y NAT, que permite a redes privadas acceder a internet.

También se incluirán prácticas con NTP para sincronización horaria, explicando por qué es crucial que todos los dispositivos de la red tengan la misma hora para el registro y análisis de incidentes. Estos laboratorios permitirán entender cómo los servicios de red interactúan con la infraestructura de enrutamiento y switching.

Módulo 5: Seguridad en la configuración de red

Este módulo introduce prácticas básicas de seguridad, como el uso de contraseñas seguras, la encriptación de contraseñas en la configuración, y la implementación de listas de control de acceso (ACL) para limitar el tráfico permitido en la red. Se enseñará cómo proteger el acceso físico y remoto a los dispositivos, evitando accesos no autorizados.

El estudiante también simulará incidentes como intentos de acceso indebido y aprenderá a identificarlos mediante registros de los dispositivos. Se fomentará la creación de políticas de respaldo y documentación que fortalezcan la seguridad de la red.

Módulo 6: Proyecto final de configuración y protocolos

Como cierre del curso, se propondrá el diseño y configuración de una red empresarial que integre todos los temas vistos. El proyecto incluirá la implementación de direccionamiento IP estructurado, enrutamiento dinámico, servicios de red y medidas básicas de seguridad.

El estudiante deberá presentar un informe técnico donde se detalle la planificación de la red, los comandos utilizados, las pruebas de conectividad realizadas y las soluciones aplicadas a problemas encontrados durante el desarrollo. Este proyecto busca consolidar las competencias adquiridas y simular un entorno profesional.

Prácticas y evaluación

Cada módulo incluye prácticas en simuladores y emuladores. El estudiante deberá entregar capturas de configuración, archivos de simulación y un breve informe técnico. La evaluación se centrará en la correcta implementación, la resolución de problemas y la claridad en la documentación presentada.

El proyecto final tendrá un peso significativo en la calificación global, ya que refleja la capacidad de integrar todos los conocimientos del curso en un escenario realista.

Recursos adicionales

El curso incluye materiales complementarios como plantillas de direccionamiento IP, ejemplos de configuraciones paso a paso, enlaces a repositorios de topologías prediseñadas y manuales de referencia de comandos. Estos recursos están pensados para reforzar el aprendizaje autónomo y servir como base en entornos laborales.
